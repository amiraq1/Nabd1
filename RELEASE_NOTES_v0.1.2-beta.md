# نبض - Nabd v0.1.2 Beta

تحسينات هذا الإصدار:
- **إضافة ميزة Ask Image:** إمكانية السؤال عن الصور واستخراج النصوص منها عبر تقنية OCR.
- **دعم FastVLM:** دعم اختياري لنموذج الرؤية FastVLM للوصف البصري المتقدم.
- **أداة الخريطة:** إضافة أداة الخريطة مع دعم البحث عن الأماكن، عرض المسارات، وحفظ الأماكن المفضلة.
- **أنماط استخراج المستندات:** إضافة خيارات متقدمة عند استخراج النصوص (نص خام، Markdown منظم، بيانات منظمة، وإزالة المعلومات الحساسة) تعمل محليًا بالكامل.
- **تحسين الرسائل:** توحيد وتحسين رسائل الخطأ باللغة العربية لتكون أكثر وضوحاً وفائدة.
- **الدليل التقني:** تحسين توثيق الأخطاء الشائعة والحلول في ملف README.
- **الخصوصية:** تحديث سياسة الخصوصية لتشمل تفاصيل استخدام ميزات الخريطة والرؤية.
- **تحديثات المعمارية:** تحسينات هيكلية في طبقة محرك الاستنتاج (Inference Engine) لزيادة الكفاءة.
- **الاستقرار:** إصلاحات متعددة لمشاكل البناء وتحسين استقرار التطبيق العام.

---

**ملاحظة هامة:**
النماذج غير مدمجة داخل ملف التطبيق (APK). يجب على المستخدم استيراد ملفات النماذج بصيغة `.litertlm` يدويًا من داخل إعدادات التطبيق ليعمل المساعد.
