# نبض - Nabd

نبض هو مساعد ذكاء اصطناعي محلي يعمل على Android بدون خادم خارجي.

## المميزات الرئيسية

- تشغيل نماذج LiteRT-LM محليًا على الجهاز
- دعم Gemma E2B للمهام اليومية
- دعم Gemma E4B للمهام الثقيلة
- محادثات متعددة مثل ChatGPT
- مكتبة مستندات محلية
- تحليل ملفات PDF والصور
- إجابات من المستندات مع مصادر
- أدوات هاتف آمنة
- Safe Agent Loop مع تأكيد المستخدم
- مركز أدوات
- صفحة إعدادات
- Material Design 3

## الخصوصية

- المعالجة تتم داخل الجهاز
- لا يتم إرسال المحادثات إلى خادم خارجي من داخل التطبيق

## المتطلبات

- جهاز Android حديث
- ذاكرة RAM جيدة، ويفضل 8GB أو أكثر للنماذج الثقيلة
- مساحة تخزين كافية لملفات النماذج

## ملاحظة حول النماذج

- النماذج غير مدمجة داخل التطبيق
- يجب استيراد ملفات `.litertlm` يدويًا داخل نبض
- لا ترفع ملفات النماذج إلى GitHub أو إلى المستودع

## النماذج المقترحة

- Gemma E2B
- Gemma E4B

## طريقة الاستخدام

1. ثبّت ملف APK.
2. افتح نبض.
3. اختر النموذج.
4. استورد ملف `.litertlm`.
5. شغّل نبض.
6. ابدأ المحادثة.

## ما الذي يمكن لنبض فعله؟

- الدردشة المحلية مع نماذج LiteRT-LM
- إدارة محادثات متعددة
- قراءة الصور واستخراج النص منها عبر OCR
- قراءة ملفات PDF وتحليلها
- حفظ المستندات محليًا واستخدامها في الأسئلة اللاحقة
- تشغيل أدوات هاتف آمنة بعد تأكيد المستخدم
