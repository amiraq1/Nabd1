# نبض - Nabd

نبض هو تطبيق أندرويد لتشغيل نماذج الذكاء الاصطناعي محليًا على الجهاز بدون خادم خارجي.

## المميزات

- تشغيل محلي بالكامل على Android
- دعم LiteRT-LM
- دعم Gemma E2B للمهام اليومية
- دعم Gemma E4B للمهام الثقيلة
- واجهة عربية
- تحليل الصور عبر OCR
- تحليل ملفات PDF
- تعدد النماذج
- حفظ المحادثات محليًا
- نسخ ولصق المحادثات
- تصميم داكن مستوحى من لوحة ألوان Rork

## الخصوصية

تتم المعالجة داخل الجهاز، ولا يتم إرسال المحادثات إلى خادم خارجي من داخل التطبيق.

## المطور

إعداد وتطوير: عمار محمد التميمي

## ملاحظة

ملفات النماذج مثل `.litertlm` غير مرفوعة إلى المستودع بسبب حجمها الكبير.  
يجب تنزيل النماذج واستيرادها داخل التطبيق.
