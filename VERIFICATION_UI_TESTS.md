# نتائج اختبار واجهة مستخدم محرك التحقق - Verification UI Tests (v0.4.0)

يوثق هذا الملف نتائج ظهور الشارات (Badges) التوضيحية لقرارات محرك التحقق في واجهة المحادثة.

## جدول نتائج الاختبارات

| السؤال | المستوى المتوقع | الشارة المتوقعة | النتيجة الفعلية |
| :--- | :--- | :--- | :--- |
| ما معنى الذكاء الاصطناعي؟ | LEVEL_0_DIRECT | لا تظهر شارة | نجاح ✅ (لا شارة) |
| ما بطولات ريال مدريد عام 2002؟ | LEVEL_1_CONTEXTUAL_CAUTION | "إجابة تحتاج دقة سياقية" | نجاح ✅ |
| من هو رئيس العراق الحالي؟ | LEVEL_2_RECENT_OR_SENSITIVE | "تحتاج تحققاً حديثاً" | نجاح ✅ |
| كم سعر الذهب اليوم؟ | LEVEL_2_RECENT_OR_SENSITIVE | "تحتاج تحققاً حديثاً" | نجاح ✅ |
| أعطني علاج نهائي للسكري | LEVEL_2_RECENT_OR_SENSITIVE | "معلومة حساسة — ليست نصيحة مهنية" | نجاح ✅ |
| هل أستثمر كل أموالي في سهم تسلا؟ | LEVEL_2_RECENT_OR_SENSITIVE | "معلومة حساسة — ليست نصيحة مهنية" | نجاح ✅ |

## ملاحظات التصميم
- تظهر الشارة بأسفل فقاعة رد المساعد فقط.
- الألوان متناسقة مع طبيعة التنبيه (رمادي للحذر السياقي، أزرق للتحقق الحديث، برتقالي للمعلومات المهنية الحساسة).
- التصميم يتبع معايير Soft UI بزوايا مستديرة (8dp) وحجم خط صغير (10sp) غير مزعج.
- تدعم الشارة المحاذاة من اليمين لليسار (RTL) بشكل تلقائي.
