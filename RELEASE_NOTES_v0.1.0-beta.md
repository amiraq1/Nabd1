# Nabd v0.1.0 Beta

## Highlights

- أول نسخة تجريبية عامة من نبض على Android
- تشغيل محلي لنماذج LiteRT-LM
- دعم مستندات وصور وPDF
- أدوات هاتف آمنة مع تأكيد المستخدم
- واجهة عربية بتصميم Material Design 3

## Features

- دعم Gemma E2B و Gemma E4B
- محادثات متعددة
- مكتبة مستندات محلية
- استخراج النص من الصور وPDF
- إجابات من المستندات باستخدام الاسترجاع المحلي
- مركز أدوات
- صفحة إعدادات مستقلة

## Known limitations

- الأداء يختلف حسب الجهاز
- ملفات PDF الكبيرة قد تكون بطيئة أو قد تفشل
- النماذج كبيرة الحجم وغير مضمنة
- Gemma E4B قد يكون أثقل على بعض الهواتف

## Installation steps

1. ثبّت ملف APK.
2. افتح نبض.
3. اختر النموذج من الإعدادات.
4. استورد ملف `.litertlm`.
5. شغّل نبض.
6. ابدأ المحادثة أو جرّب تحليل مستند.

## Model import note

- النماذج غير مدمجة داخل التطبيق أو داخل المستودع
- يجب استيراد ملفات `.litertlm` يدويًا
- لا ترفع ملفات النماذج إلى GitHub Releases أو إلى المستودع

## Privacy note

المعالجة تتم داخل الجهاز، ولا يتم إرسال المحادثات إلى خادم خارجي من داخل التطبيق.
