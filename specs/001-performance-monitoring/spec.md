# Feature: Performance Monitoring UI

## Goal
عرض مؤشرات أداء النموذج المحلي داخل نبض أثناء الاستخدام.

## Requirements
- عرض TPS الحالي.
- حفظ آخر 20 قراءة TPS.
- عرض RAM الحالي.
- حفظ آخر 20 قراءة RAM.
- تحديث RAM كل ثانيتين فقط.
- عرض حالة فارغة مفهومة عند عدم وجود بيانات TPS.
- عدم إرسال أي بيانات خارج الجهاز.
- عدم التأثير على تدفق المحادثة.
- نجاح بناء Debug.

## Acceptance Criteria
- BUILD SUCCESSFUL.
- لا يحدث crash عند فتح Bottom Sheet.
- تظهر رسالة فارغة عند عدم وجود TPS.
- يتم تحديث RAM بدون تهنيج.
- لا يتم حفظ أكثر من 20 قراءة.
