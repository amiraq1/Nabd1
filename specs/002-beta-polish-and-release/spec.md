# Feature: Beta Polish and Release Readiness

## Goal
تجهيز نسخة نبض Beta للاختبار اليدوي والنشر كتجربة عامة.

## Requirements
- تحديث CHANGELOG أو Release Notes.
- التأكد من أن Performance UI لا يكسر المحادثة.
- اختبار فتح Bottom Sheet.
- اختبار المحادثة بدون نموذج.
- اختبار المحادثة مع نموذج مستورد.
- اختبار PDF/RAG.
- اختبار OCR.
- اختبار الخريطة.
- نسخ APK النهائي إلى Download.
- توثيق نتيجة الاختبار.

## Acceptance Criteria
- BUILD SUCCESSFUL.
- APK موجود في /sdcard/Download.
- git status نظيف.
- لا توجد ملفات مكررة أو untracked غير مقصودة.
