# نبض - Nabd v0.2.0 Beta

تحسينات هذا الإصدار:

- **مراقبة الأداء (Performance UI):** إضافة واجهة لمراقبة سرعة الاستجابة (TPS) واستهلاك الذاكرة (RAM) لحظياً من خلال نافذة الإعدادات السريعة.
- **تأمين البيانات والخصوصية:** 
    - تشفير جميع التفضيلات المحلية باستخدام AES-256.
    - تشفير قاعدة البيانات المحلية باستخدام SQLCipher.
    - إضافة التحقق من سلامة النماذج (SHA-256).
    - إزالة أذونات الشبكة بالكامل لضمان الخصوصية القصوى.
- **تحسينات الواجهة (UI/UX):**
    - إضافة مؤشر تحميل تفاعلي (EKG indicator).
    - توسيع مساحة الدردشة وتحسين توزيع العناصر.
    - تحسين استقرار تدفقات إعدادات RAG.
- **استقرار النظام:**
    - تثبيت إصدارات جميع المكتبات البرمجية لضمان توافقية البناء.
    - تحسين التعامل مع تقارير التشخيص لإخفاء مسارات النظام الحساسة.
- **إصلاحات عامة:** معالجة مشاكل البناء واستقرار محرك الاستنتاج.

---

**ملاحظة هامة:**
هذا الإصدار يركز بشكل كبير على الأمان والخصوصية. تم تشفير قاعدة البيانات، لذا قد يتطلب الأمر مسح بيانات التطبيق إذا كنت تستخدم إصداراً قديماً جداً (قبل v0.1.2) لضمان التوافق مع نظام التشفير الجديد.
