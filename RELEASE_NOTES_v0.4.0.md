# v0.4.0 - Sources & Verification Engine

## أبرز الميزات
- إضافة Verification Engine لتصنيف أسئلة المستخدم قبل الإجابة.
- إضافة VerificationClassifier لتحديد مستوى التحقق المطلوب.
- إضافة VerificationPromptBuilder لحقن تعليمات تحقق مخصصة داخل البرومبت.
- دعم 3 مستويات تحقق:
  - Level 0: إجابة مباشرة.
  - Level 1: دقة سياقية للأسئلة التاريخية والرياضية.
  - Level 2: تحقق حديث أو تعامل آمن مع المواضيع الحساسة.
- إضافة شارات تحقق داخل واجهة المحادثة:
  - "إجابة تحتاج دقة سياقية"
  - "تحتاج تحققاً حديثاً"
  - "معلومة حساسة — ليست نصيحة مهنية"
- تحسين التعامل مع:
  - الأخبار والأسعار.
  - المسؤولين الحاليين.
  - الأسئلة الطبية والمالية والقانونية.
  - البطولات والتواريخ والمواسم الرياضية.
- تقليل الهلوسة ومنع الجزم في المعلومات المتغيرة دون تحقق.

## نتائج الاختبار
- متوسط التقييم: 4.95/5
- نجاح تصنيف 10/10 حالات اختبار.
- عدم ظهور شارات في أسئلة Level 0.
- نجاح الشارات في الأسئلة الحساسة والتاريخية.
- لا توجد هلوسة واضحة في الاختبارات الأساسية.
- لا توجد نصائح طبية أو مالية حاسمة.
